Cookie Collector 3DS

Kaisogen

Cookie Collector 3DS was made in 2016 as a test of 3DS Homebrew Libraries. You can download the original here.

Alternatives

You can use the source as a way of learning a bit of C with the 3DS, but I would not play it. More features are being added by the day, so keep a look out! Alternatives to this game includes:

    Creckeryop's Cookie Clicker
    TheMachinumps' Cookie Clicker 3DS
    TheMachinumps' Cookie Collector 3DS Fork

Usage

Grab a release file from the releases page, or compile the code yourself. Place the 3DSX and SDMH in /root/3DS/games/CookieCollector/

Your files should look like this:

/root 
    /3DS
            /games
            /CookieCollector
                CookieCollector.3DSX
                CookieCollector.SDMH

Run the CookieCollector.3DSX file in the Homebrew Launcher, from an entrypoint of your choice. Have fun.

Compilation

Clone the repo, and unzip it somewhere. Make a new directory labelled 'source', and place 'main.c' in there. CD to the root of the repo, and run 'make'. Once it has finished, it will produce an ELF and a 3DSX file of the same name. Grab the SDMH from the releases page, and then use that instead of the generated one.

This assumes you have installed the necessary libraries and DevkitPro.
